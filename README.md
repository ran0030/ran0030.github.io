# Melbourne Public Transport Data Analysis

This project analyses public transport patronage trends in Melbourne from 2018–2024.

The task was completed for VCE Computing Unit 1/2 (S1 AT1) and investigates how usage changed over time, particularly during and after COVID-19, comparing metropolitan and regional services.

The website presents data visualisations, statistical analysis, and conclusions based on official Victorian transport data.
